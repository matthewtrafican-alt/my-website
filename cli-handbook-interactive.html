Text & Output

Piping & FilteringChain and transform output▾
# Filter by condition
ls | Where-Object { $_.Length -gt 1MB }
ls | Where-Object Name -like "*.log"
# Sort and select
ls | Sort-Object Length -Descending | Select-Object -First 10
# Export
ls | Export-Csv output.csv -NoTypeInformation
ls | ConvertTo-Json
# Redirect output
command > output.txt           # Stdout to file
command 2> errors.txt          # Stderr to file
command > out.txt 2>&1        # Both to same file
command | Tee-Object out.txt  # Print AND save

Environment Variables

Read, set, and persist variables▾
$env:PATH                           # Read a variable
Get-ChildItem Env:                  # List all
$env:MY_VAR = "hello"              # Set (session only)
$env:PATH += ";C:\new\path"       # Append to PATH
[System.Environment]::SetEnvironmentVariable("MY_VAR","value","User")  # Persistent
# CMD equivalents
set MY_VAR=hello  |  setx MY_VAR "hello"  |  echo %MY_VAR%
